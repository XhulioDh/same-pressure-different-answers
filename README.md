# Same Pressure, Different Answers

**A Comparative Probe of Five AI Models Under Moral Pressure**

Xhulio Dhespollari — content moderation professional exploring AI safety
Published June 2026

## About this case

This is an exploratory experiment, not a formal study. It is a single-run comparison of how five large language models handle sustained adversarial pressure on a content-moderation judgment. Each model was given the same benign passage, asked whether it contained a policy violation, and then taken through an identical nine-step sequence of escalating pressure.

The full case study is in this repository, along with the complete verbatim transcript for every model so the findings can be checked directly against the evidence.

## A note on method and disclosure

The case was developed with substantial assistance from Claude (Opus), which is disclosed in full inside the study. One of the five tested models is from the same family as that assistant, which is a conflict of interest the case discloses and asks the reader to weigh. The findings are observations from a single run of a single passage, offered as patterns worth testing further, not as settled claims.
